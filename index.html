VERIFICA INFOMATICA

DOMANDA SU FILE DI TESTO E BINARI

I file di testo sono String, quelli binari sono byte. I file testo testo si usano perchè sono più veloci in letura e ordinamento, i file binari si usano perchè più veloce in modifica

DOMANDA LETTURA E SCRITTURA BUFFERIZZATA

con lettura e scrittura bufferizzata si intende eseguire l'azione di lettura e scrittura utilizzando il buffer, vengono praticamente messi degli elementi del file in memoria centrale. Migliora la velocità perchè non va a leggere dal file ma direttamente dal buffer. Peggiora però nel caso si voglia tornare indietro in questo caso infatti il buffer dovrà tornare indietro, quest'azione farà perdere del tempo.

DIFFERENZA TRA CANCELLAZIONE LOGICO E FISICO

Logica ricicla spazio ed è più veloce nella cancellazione dato che nasconde, fisica occupa meno spazio ed elimina subito.

COME COPIARE UN FILE IN UN ALTRO FILE

creo un file di supporto oltre al filfile originale.Nel momento in cuicopio gli elementi da un file all'altro controllo che la stringa passata non contenga quella che intendo eliminare e, se la stringa corrisponde, non la copio nel file di supporto. Una volta ricopiato il file, lo ricopio nel file originale.
